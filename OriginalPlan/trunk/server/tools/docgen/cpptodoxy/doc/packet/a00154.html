enum EPacketIDDef

消息ID定义枚举

消息ID值描述格式：PACKET_XX_YYYYYY XX可以描述为：MC、CM、MW、WM、CW、WC、MR、WR、PS、SP M游戏服务器端、C客户端、W世界服务器端、R为记录服务器、MG表示管理服务器 YYYYYY表示消息内容 例如：PACKET_CM_ATTACK 表示客户端发给游戏服务器端关于攻击的消息消息ID定义枚举
